**Metadatos:**
- Título del lanzamiento: **Maat encarnada** · Tipo: **Single**
- Artista principal: **Carlos Romero** (si quieres acreditar a la cantante, añádela como *featured*)
- Idioma: **Español** (voz femenina cantada)
- Género: **Jazz** → subgénero **Bossa Nova** (es fusión, casi bossa nova). Alternativa: *Latin*
- (P) y (C): **2026 Carlos Romero**
- Pista 1: *Maat encarnada* · Compositor: *Carlos Alfonso Romero Muñoz* · Cover: **No** · Explícito: **No**
- ISRC: **vacío** (RouteNote lo genera gratis)
- Fecha de lanzamiento: **a 2–3 semanas** (margen para aprobación de tiendas)
- Tiendas: marcar **todas** las del plan free

### Pasos en RouteNote
1. Login → **Distribution** → **Create New Release**.
2. Tipo: **Single**.
3. Rellena datos del release (arriba).
4. Sube la **portada** (`...portada 3000.jpg`).
5. Sube el **audio** (`Maat encarnada.wav`); completa título, compositor, Cover=No, Explicit=No, ISRC vacío.
6. Marca **todas** las tiendas.
7. **Review → Submit/Distribute** y acepta términos (confirmas que es composición tuya).

### Nota de calidad
El WAV proviene de un MP3 ~164 kbps (no hay máster original). Pasa el validador,
pero si consigues una exportación mejor, conviene resubir.
